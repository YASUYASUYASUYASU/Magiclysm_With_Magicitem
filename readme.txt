○Magiclysm アドオン

	[追加 - Magiclysm アドオン]
	(MOD識別名：magiclysm_addon)


	【概要】
	・魔法modことMagiclysmのアドオンになります。
	　様々な魔法アイテムを追加し、魔法使いをサポートするmodになります。
	　魔法の銃では各種ルーンを素材に弾丸を作る事が可能で、
	　素材にしたルーンによって弾丸の効果が変わります。
	　・ケルビニスト⇒確率で炎上させたり、敵の動きを止めたりする
	　・アースシェイパー⇒着弾時に弾丸が炸裂し、周囲に破片ダメージ
	　・ストームシェイパー⇒周囲にダメージが連鎖
	　・ドルイド⇒着弾時に相手を盲目にする
	　など
	https://i.imgur.com/nZN8V9D.png
	https://i.imgur.com/SbeoiUZ.png
	

	魔法武器までのつなぎや予備武器としてどうぞ
	また、変異しても装着できる魔法アクセサリも追加されます。
	Magiclysmの指輪と同じ場所で見つけることができますが、
	生成率が本来の指輪と比べて低めとなってます。
	※このmodのアイテムはアイテム名の末尾に[MA]と付きます
	まだアイデア的に網羅しきれてないルーンがありますが、
	今後検討していこうかと思います。
	また、複数のルーンを組み合わせた上位の弾丸も実装予定です

	今後このmodでは、魔法使いをサポートする
	Magiclysmには無いアイテムを追加していく予定です

	
	◆導入方法
	ダウンロードしたzipファイルを解凍して
	modフォルダに配置して頂ければOKです。
